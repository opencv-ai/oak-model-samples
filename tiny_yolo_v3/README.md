## Tiny-YOLOv3

This is an inference code to run a Tiny-YOLOv3 model on DepthAI using Gen2 Pipeline Builder.

The original model could be found in PINTO Model Zoo [there](https://github.com/PINTO0309/PINTO_model_zoo).
